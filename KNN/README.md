# 🩺 Diabetes Prediction using KNN

A simple and effective Machine Learning project that predicts whether a person is **diabetic or not** using the **K-Nearest Neighbors (KNN)** algorithm.

---

## 🚀 Features

* 📊 Data preprocessing & cleaning
* 🔍 Exploratory Data Analysis (EDA)
* 🤖 KNN model for classification
* 📈 Model evaluation (accuracy, confusion matrix)
* 🔮 Real-time prediction using custom input

---

## 🧠 Tech Stack

* Python 🐍
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

---

## 📁 Dataset

* Pima Indians Diabetes Dataset
* Features include: Glucose, BMI, Age, etc.

---

## ⚙️ How It Works

1. Clean and preprocess the dataset
2. Perform EDA to understand patterns
3. Apply feature scaling
4. Train KNN model
5. Predict diabetes outcome

---

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python main.py
```

---

## 🔮 Sample Prediction

Input:

```
[2, 120, 70, 20, 79, 25.0, 0.5, 30]
```

Output:

```
Not Diabetic ❌
```
