# Advanced Ensemble Learning with XGBoost

## 📌 Project Overview

This project demonstrates the implementation and comparison of multiple machine learning models along with advanced ensemble techniques. The goal is to improve prediction performance using methods like Voting, Stacking, Hybrid Ensemble, and XGBoost.

---

## ⚙️ Technologies Used

* Python
* Scikit-learn
* XGBoost
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📊 Models Implemented

* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest
* Gradient Boosting
* K-Nearest Neighbors (KNN)
* XGBoost

---

## 🔥 Ensemble Techniques Used

* Voting Classifier (Soft Voting)
* Stacking Classifier
* Hybrid Ensemble (Voting + Boosting)

---

## 🧠 Workflow

1. Data Preprocessing (Handling missing values, scaling)
2. Handling imbalanced data using SMOTE
3. Training multiple machine learning models
4. Evaluating models using:

   * Accuracy
   * Precision
   * Recall
   * F1 Score
   * ROC-AUC
5. Applying ensemble techniques
6. Comparing performance of all models

---

## 📈 Results

* Ensemble models improved overall performance
* XGBoost provided strong results compared to traditional models
* Hybrid models helped balance precision and recall

---

## ▶️ How to Run

1. Clone the repository
2. Install required libraries:

   ```
   pip install -r requirements.txt
   ```
3. Open the notebook:

   ```
   advanced-ensemble-learning.ipynb
   ```

---

## 🎯 Conclusion

Ensemble learning techniques significantly improve model performance by combining multiple models. XGBoost and Hybrid ensembles provided the best results in this project.
